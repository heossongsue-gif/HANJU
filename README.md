# 🍶 HANJU - 한주 투어 관리 시스템
---

## 프로젝트 개요 (プロジェクト概要)
- **개발 기간:** 2026.03 (약 1주일)
- **주요 목적:** 수기 및 메신저로 관리하던 가이드 투어 예약과 고객 정보를 데이터베이스화하여 관리 효율 증대
- **배포 주소:** [https://hanju.vercel.app](https://hanju.vercel.app)

## 주요 기능 (主な機能)
- **🔐 인증 시스템:** **Supabase Auth**를 활용한 안전한 소셜 및 이메일 로그인 구현
- **📂 실시간 데이터 관리:** **Supabase DB**를 연동하여 고객별 폴더 분류 및 상세 정보를 실시간으로 저장 및 관리
- **📅 투어 스케줄링:** 투어 기간 및 일정 최적화 관리 로직 구현
- **📱 반응형 UI:** **Tailwind CSS**를 사용하여 현장에서도 편리하게 사용할 수 있는 모바일 최적화 레이아웃 제공

## 기술 스택 (기술 스택 & 인프라)
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Backend / Database:** **Supabase** (Auth, Database)
- **Styling:** Tailwind CSS
- **Deployment:** **Vercel** (CI/CD 자동화 및 호스팅)

## AI Native Development & Efficiency
본 프로젝트는 최신 AI 에디터인 **Cursor**를 전략적으로 활용하여 개발되었습니다.
- **Full-Stack 개발:** AI를 활용해 프론트엔드부터 백엔드(Supabase 연동)까지의 전체 아키텍처를 빠르게 설계
- **CI/CD 환경 구축:** **Vercel**과의 연동을 통해 코드 수정 시 자동 배포되는 환경을 구축하여 개발 생산성 극대화
- **검증 및 최적화:** AI가 제안한 로직을 직접 분석하고 에러를 디버깅하며, 실제 사용자(어머니)의 피드백을 즉각 반영함
